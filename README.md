#  Secure Notes App

A simple Flutter based note taking application focused on privacy and simplicity develop as my semester project.
This app allows users to securely store personal notes locally on their device using a PIN-based authentication system.This application demonstrates how to build a lightweight secure note system without relying on external servers.All user data is stored locally, making it fast, offline-capable, and privacy-friendly.


---

## 📱 Features

*  PIN Authentication (first-time setup + login)
*  Create notes with title and content
*  Automatic timestamp for each note
*  Delete notes
*  Local storage using SharedPreferences
*  Clean dark-themed UI
